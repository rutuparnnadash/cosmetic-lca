
# Cosmetic Product LCA Calculator

This is a Streamlit web application to calculate the estimated life cycle carbon emissions (COâ‚‚-eq) of a cosmetic product, based on its formulation, packaging, and usage.

## How to Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```

## How to Deploy on Streamlit Cloud

1. Push this repository to GitHub.
2. Go to https://streamlit.io/cloud and link your GitHub account.
3. Select this repo and deploy.

Enjoy!
